# 카페 근무표 자동생성기 — 사용방법

> **준비물:** Chrome 또는 Edge 브라우저 (Safari는 일부 기능 미지원)

---

## 시작하기

1. `cafe-scheduler` 폴더 안의 **`index.html`** 파일을 더블클릭해서 엽니다.
2. 크롬/엣지가 자동으로 앱을 실행합니다. (인터넷 연결 불필요)

---

## 처음 한 번만 — 초기 설정

### 1. 직원 이름 입력

왼쪽 사이드바 **"직원 세팅"** 항목을 펼쳐서 직원 이름을 입력합니다.

- 직원1, 직원2, 직원3, 직원4 칸에 실제 이름을 씁니다.
- 이름 옆 **☆ 별 버튼**을 누르면 ⭐로 바뀌면서 **주말·공휴일 마감 선호** 직원으로 등록됩니다.
  → 일정 생성 시 해당 직원이 주말·공휴일에 마감(C)으로 배치되도록 우선 고려합니다.

### 2. 데이터 파일 연결 (중요!)

왼쪽 사이드바 맨 아래 **"데이터 파일 저장"** 항목을 펼칩니다.

- **"✨ 새 파일 만들기"** 버튼 클릭
- 파일 이름을 `cafe-scheduler-data.json`으로 하고, **`cafe-scheduler` 폴더 안에 저장**

> 이렇게 하면 이후 수정사항이 모두 이 파일에 자동으로 저장됩니다.
> 다음에 앱을 열면 자동으로 불러옵니다.
> (처음 열 때 "파일 접근 허용" 팝업이 뜨면 **허용**을 눌러주세요)

---

## 매주 하는 일 — 근무표 만들기

### 1. 사이클 선택

화면 상단 **◀ ▶ 버튼**으로 원하는 4주 기간을 선택합니다.
**"오늘"** 버튼을 누르면 현재 기간으로 바로 이동합니다.

### 2. 배정 요청 등록 (선택)

특정 직원이 특정 날 오픈/마감/휴무를 원하면 **"배정 요청"** 항목에 미리 등록합니다.

1. 직원 선택 → 날짜 선택 → 오픈/마감/휴무 선택
2. **"추가"** 버튼 클릭

> 등록된 요청은 일정 생성 시 **고정 배정**으로 반영됩니다.

### 3. 일정 생성

**"이 사이클 일정 생성"** 버튼을 클릭합니다.

- 자동으로 4주 근무표가 만들어집니다.
- 하드 조건(왼쪽 패널)이 모두 ✅이면 정상입니다.
- ⚠ (노란색)은 고정 배정 때문에 어쩔 수 없이 어긋난 경우이므로 무시해도 됩니다.
- ❌ (빨간색)이 있으면 다시 생성해보세요. (같은 버튼을 다시 클릭)

### 4. 수동 수정 (필요한 경우)

생성된 근무표 셀을 직접 클릭해서 수정할 수 있습니다.

| 동작 | 결과 |
|------|------|
| **좌클릭** | 오픈(O) → 마감(C) → 휴무 순서로 변경 |
| **우클릭** | 고정 배정 메뉴 (오픈 고정 / 마감 고정 / 휴무 고정 / 해제) |

> 고정 배정된 셀은 다음번 일정 생성 때도 그 값을 유지합니다.

## 공휴일 등록

왼쪽 사이드바 **"공휴일 관리"** 항목을 펼쳐서 추가합니다.

- 날짜 선택 → 공휴일 이름 입력(예: 광복절) → **"추가"** 클릭
- 등록된 공휴일은 근무표에서 빨간색으로 표시됩니다.

---

## 내보내기 (근무표 공유)

화면 오른쪽 상단 버튼으로 파일을 내보낼 수 있습니다.

| 버튼 | 용도 |
|------|------|
| 🖼 **이미지** | 카카오톡 등에 바로 공유할 PNG 이미지 |
| 📄 **CSV** | 엑셀에서 열 수 있는 기본 표 |
| 📊 **엑셀** | 색상·스타일이 적용된 엑셀 파일 (.xlsx) |
| 💾 **백업 파일 저장** | 설정·배정 데이터 전체를 JSON으로 백업 |
| 📂 **백업 파일 열기** | 백업한 JSON 파일 복원 |

---

## 다른 사람에게 공유하는 방법

1. `cafe-scheduler` 폴더 전체를 압축(zip)해서 전달합니다.
   - 폴더 안에 `cafe-scheduler-data.json`이 있으면 데이터도 함께 전달됩니다.
2. 받는 사람은 압축을 풀고 `index.html`을 크롬/엣지로 열면 됩니다.
3. 처음 열 때 **"데이터 파일 저장"** 항목에서 **"📂 파일 연결"** 클릭 →
   `cafe-scheduler-data.json` 파일을 선택합니다.

## 자주 묻는 질문

**Q. 앱을 다시 열었더니 "파일 접근 허용" 팝업이 떴어요.**
→ **허용**을 누르면 됩니다. 브라우저 보안 정책으로 파일을 열 때마다 한 번씩 확인합니다.

**Q. 데이터가 사라졌어요.**
→ 브라우저를 바꾸거나 캐시를 삭제하면 로컬스토리지 데이터가 없어질 수 있습니다.
   파일 연결을 해두면 파일에 저장되어 있으니 "파일 연결"로 다시 불러오세요.

**Q. 일정을 다시 생성하면 수동으로 고친 것도 초기화되나요?**
→ 고정 배정(우클릭으로 잠근 것)은 유지됩니다.
   일반 셀을 좌클릭으로 바꾼 것은 다시 생성하면 초기화됩니다.

**Q. ❌가 계속 나와요.**
→ 같은 버튼을 2~3번 더 눌러보세요. 매번 다른 조합으로 최적을 찾습니다.
   고정 배정이 너무 많으면 조건을 만족하기 어려울 수 있습니다.
